# SoFlowUnchained
Android app for SoFlow SO2 Air Gen 2. (like the Grover Edition). Connects via Bluetooth Low Energy to send predefined hex commands. Buttons for 20/27 km/h, ECO, Normal, Sport, and Lock/Unlock. 100% local, no server, no tracking. For tinkerers only â€“ use at your own risk.

## Development Setup

This repository excludes the Gradle wrapper JAR from version control.
Run `gradle wrapper` to generate `gradle/wrapper/gradle-wrapper.jar` before building.
